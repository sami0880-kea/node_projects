Export / Import

With Node.js we can export and import modules. There are 2 different type of modules in Node.js, CommonJS and ES Module. These two are a bit different in how you structure your export and imports. We can change which type we use by changing our package.json. We can add the code below to change to ES Module. 

            
  {
    "type": "module" // or "commonjs"
  }
In this course we tend to use ES Module instead of the default module in Node.js

CommonJS

CommonJS is the default module for Node.js projects. In CommonJS when we import modules we are using const express = require('express'); and when we export we use module.exports = { express };

ES Modules

We are able to change our module to ES Module as mentioned earlier. ES stands for ECMAScript. It is a bit different from CommonJS. When we import modules we use import express from 'express'; and when we export we use export function sayHello() {return "Hello"}
